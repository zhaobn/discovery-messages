# Discovery Messages

A behavioral experiment studying how people use social information (messages from previous players) to guide learning and decision-making in a resource discovery task.

## Repository Structure
```
discovery-messages/
├── experiment/     # Browser-based experiment (HTML/CSS/JS/PHP)
├── python/         # Data processing, sampling, and (later) computational modeling
├── data/           # Raw and processed data
└── R/              # Statistical analysis (coming soon)
```

## Experiment

A single-page experiment built in plain HTML/CSS/JS.

## Python

Data preprocessing scripts, including message sampling pipeline.
